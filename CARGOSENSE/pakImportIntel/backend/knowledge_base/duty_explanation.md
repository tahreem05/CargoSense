# Duty Explanation: Understanding Taxes on Imports

Pakistan's import tax structure is complex and includes several components:

1. **Customs Duty (CD):** 
   The primary tariff applied to the CIF (Cost, Insurance, Freight) value. Rates vary by HS code (e.g., 0%, 3%, 11%, 20%).

2. **Regulatory Duty (RD):** 
   An additional duty imposed to discourage luxury imports or protect local industries. Can range from 10% to over 100%.

3. **Additional Customs Duty (ACD):** 
   A flat percentage (often 2% to 7%) applied to the dutiable value (CIF + CD + RD).

4. **Sales Tax (ST):** 
   A value-added tax applied at the import stage, standard rate is 18%, calculated on (CIF + CD + RD + ACD).

5. **Withholding Tax (WHT):** 
   An advance income tax collected at import. Filers (active taxpayers) typically pay 5.5%, while non-filers pay higher rates like 8% or more.
