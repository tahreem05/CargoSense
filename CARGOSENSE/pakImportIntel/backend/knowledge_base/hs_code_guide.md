# HS Code Guide: Finding the Right Classification

Harmonized System (HS) codes are an internationally standardized system of names and numbers to classify traded products.

1. **Structure of an HS Code:**
   - **Chapter:** First 2 digits (e.g., 85 - Electrical machinery)
   - **Heading:** Next 2 digits (e.g., 85.17 - Telephones)
   - **Subheading:** Next 2 digits (e.g., 8517.13 - Smartphones)
   - **Tariff Item (Pakistan specific):** Additional 2 digits defining specific national variations.

2. **Why it Matters:**
   The HS code dictates the duty rates (CD, RD, ACD) applied to your import. Using the wrong code can lead to heavy fines or delays if customs suspects misdeclaration.

3. **How to Find Your HS Code:**
   - Consult the Pakistan Customs Tariff PDF (FBR).
   - Ask your supplier for the global 6-digit HS code and map it to the Pakistan 8-digit system.
   - Use WCO's database or PRAL's online lookup.
